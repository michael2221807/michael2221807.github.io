Tech Stacks - JavaScript, React, JSON, NodeJS, MongoDB
Links - Pokemon Trainer

Planning

The pokemon trainer is a interactive web application that allow players to own and grow their pokemons. Every player would start with Pikachu and by playing with him, you would grant credit which can be used to purchase more pokemon. The pokemons would evolve when they reach the designed level, and will produce more credit. The frontend of the app would be build with React and Material-ui, and the backend would be built with NodeJS with express. The server would connect to the database and provide APIs for the frontend. JSON format would be used for database modeling and MongoDB would be selected.
